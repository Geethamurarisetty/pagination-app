# Pagination App

## 📌 Project Overview

The **Pagination App** is a web application built using **HTML, CSS, and JavaScript** that demonstrates how large datasets can be displayed efficiently using pagination techniques.

This project shows how users can navigate through multiple pages of data using **Next, Previous, and Page Navigation buttons** with a modern and responsive UI.

---

## 🚀 Features

* 📄 Page Navigation
* ⬅ Next and Previous Buttons
* 📊 Dynamic Data Rendering
* 🌙 Dark Mode Toggle
* 🎨 Modern UI Design
* ⚡ Smooth Animations
* 📱 Responsive Layout

---
